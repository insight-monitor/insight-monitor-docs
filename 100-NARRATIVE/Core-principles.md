# Core Principles

- **Transparency.** What is collected, why, and what is inferred from it — all visible to the monitored person at all times.
- **Challengeability.** Every conclusion has an evidence trace and can be disputed. The system learns from challenges.
- **Proportionality.** Data collection is justified per inference need. No indiscriminate logging.
- **Individual benefit.** The monitored person must gain from the system's presence. Without this, the system is not Insight Monitor.
- **Uncertainty as a feature.** The system reports what it does not know. Confidence scores are not optional.
