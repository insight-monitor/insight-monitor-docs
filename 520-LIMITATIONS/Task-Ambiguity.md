# Task Ambiguity

The same surface signals can correspond to radically different tasks.

## Examples
- YouTube: tutorial, entertainment, research, background music
- Excel: data entry, analysis, personal finance, busywork
- Slack: collaboration, socializing, waiting for input
- Browser with documentation: learning, reference, procrastination

## Limit
Without explicit task definitions from the user or integration with project management tools, the system cannot reliably distinguish these cases. Classification accuracy is bounded by the ambiguity inherent in surface signals.
