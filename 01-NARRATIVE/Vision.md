# Vision

A world where behavioral data is interpreted through context rather than surface activity.

Organizations and individuals understand work patterns, focus, and task progression using evidence-based probabilistic models instead of static productivity categories.

Insight Monitor provides the infrastructure to generate this contextual behavioral intelligence.

**Customers** — primary: organizations seeking nuanced visibility beyond binary scores; secondary: individuals tracking personal productivity. [[Market]]

**Data approach** — hybrid on-device and cloud processing with sensitive redaction before sync. Collection is justified per inference need, not indiscriminate. [[Data-Strategy]]

**Product model** — an inference pipeline (managed API) and an application layer (dashboards, integrations). Both are first-class products. [[Product-Architecture]]

**Boundaries** — limited to monitored-device signals. Multi-device, analog, and purely cognitive work is partially visible at best. [[Observability-Gaps]]
