# Use Case: Personal Productivity

## Scenario
An individual uses the system voluntarily to understand their own work patterns.

## What the system provides
- Personal time breakdown: deep work vs admin vs distractions
- Context-switching frequency and cost
- Peak productivity hours identification
- Interruption analysis
- All data stays local by default (no employer access)

## Key difference from enterprise use
The user is the sole consumer of the data. No manager dashboard. Full deletion rights. No mandatory collection.
